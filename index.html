01 · diff from parent

Eight atomic changes.

Designed to land in a single migration pass. Every change is small, surgical, and applicable across all pages without restructuring layouts or rewriting components.

01
Background.

Page becomes #FAFAF7 desk. Containers become pure white.

02
Cards float.

Drop borders, drop radius. Replace shadow with the paper-float stack.

03
Hairlines.

Every divider becomes 1px ink 10%. Drop the rest.

04
Sans only.

Space Grotesk display · Inter body · JetBrains Mono stamps. Serif removed.

05
Two greys.

Ink #0E0E0E, mute #6E6E73. No others.

06
Teal swipes.

Teal is only ever a highlighter gradient. Never a fill or a border.

07
Hard buttons.

Solid black or thin-bordered white. Mono caps label. No other variants.

08
No emoji.

The teal dot does any work emoji would do elsewhere. Functional icons stay.

05 · migration

Hand this to your engineer.

One prompt. Eight changes. No structural edits. Drops into any existing Superiora page and produces the new feel in a single pass.

Prompt · paste into your coding agent 
1.  Replace every page-level neutral background with #FAFAF7 (the desk).
    Set every card/container surface to #FFFFFF. Drop other tints.

2.  Cards become floating paper. Drop borders. Drop radius (max 6 px).
    Replace shadow with:
        0 1px 2px rgba(14,14,14,.04),
        0 8px 20px -6px rgba(14,14,14,.08),
        0 24px 40px -18px rgba(14,14,14,.12).

3.  Every divider becomes a single hairline: 1px solid rgba(14,14,14,.10).
    Remove dividers that aren't separating real sections.

4.  Type stack:
        Display → Space Grotesk · 500 · -0.02 em
        Body    → Inter
        Stamps  → JetBrains Mono · 11/9.5 px · uppercase · 0.14 em
    Remove every serif.

5.  Colors: ink #0E0E0E, mute #6E6E73. Strip all other greys.

6.  Teal #4A8580 is only ever a highlighter gradient behind inline text —
    never a flat fill, button background, badge fill, or border.

7.  Buttons: primary = ink fill, white text, no radius beyond 6 px;
    secondary = white fill, 1.25 px ink border. Mono caps label. Remove other variants.

8.  Remove all emoji. Keep functional icons (recolor to ink).
    Decorative icons next to headings become a single teal dot.
